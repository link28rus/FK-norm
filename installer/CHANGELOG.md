# История изменений установщика

## Версия 2.0.0 - MSI установщик (WiX Toolset)

### Изменения
- Заменен Inno Setup на WiX Toolset 4.x
- Установщик теперь в формате MSI
- Улучшена интеграция с Windows (стандартный интерфейс установки)
- Автоматическая генерация списка файлов через Heat

### Файлы
- `NormTracker.wxs` - основной файл конфигурации WiX
- `NormTracker.wixproj` - проект для сборки MSI
- `build-msi.ps1` - скрипт сборки MSI установщика

### Удалено
- `setup.iss` - конфигурация Inno Setup (больше не используется)
- `build-installer.ps1` - скрипт сборки Inno Setup (заменен на build-msi.ps1)
